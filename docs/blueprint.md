# **App Name**: Mastery Tracks

## Core Features:

- Curated Learning Paths: Organize diverse resources like videos, articles, and exercises into structured, goal-oriented paths, creating comprehensive educational journeys.
- Personalized AI Tutor: An AI tutor tool to assess your weak areas in Javascript by giving you tailored recommendations for learning.
- Interactive Code Challenges: Practice and apply acquired knowledge through interactive coding exercises.
- Progress Tracking: Monitor your advancement within each learning path.
- Offline Access: Enable learning on the go by providing access to downloaded content without requiring an internet connection.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), embodying knowledge, introspection, and sophistication.
- Background color: Soft lavender (#E6E6FA), providing a calming and focused learning environment.
- Accent color: Golden yellow (#FFD700), symbolizing enlightenment, achievement, and highlighting key interactive elements.
- Headline font: 'Space Grotesk' (sans-serif), offering a tech-forward, clear, and modern appeal.
- Body font: 'Inter' (sans-serif), ensuring optimal readability with a neutral and clean aesthetic.
- Consistent use of outline icons, symbolizing clarity and ease of understanding.
- Maintain a distraction-free layout, with prominent resource categories and clear calls to action.
